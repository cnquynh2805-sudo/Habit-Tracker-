# HABIT TRACKER PRO - TEST CASE DOCUMENT

## 1. Quy ước

| Ký hiệu       | Ý nghĩa                                     |
| ------------- | ------------------------------------------- |
| P1            | Bắt buộc, cần chạy trước khi demo           |
| P2            | Nên có, giúp app ổn định hơn                |
| P3            | Optional / Stretch                          |
| Positive      | Kiểm thử luồng đúng                         |
| Negative      | Kiểm thử lỗi hoặc dữ liệu sai               |
| Boundary      | Kiểm thử giá trị biên                       |
| Derived State | Kiểm thử dữ liệu được tính toán từ raw data |
| Persistence   | Kiểm thử dữ liệu không mất sau refresh      |
| Not Run       | Chưa chạy                                   |
| Pass          | Đạt                                         |
| Fail          | Không đạt                                   |
| Blocked       | Bị chặn do thiếu feature hoặc lỗi phụ thuộc |
